### BRS2(KC1) - Methyl Donor Pool

### 1. Ambition

Maintain one-carbon donor sufficiency to support remethylation, methyl-group transfer, and methylation-dependent system stability.

### 2. Shared Biological Pool

- Folate ← leafy greens, legumes, liver
- Choline ← eggs, liver, salmon
- Betaine (TMG) ← beetroot, spinach, quinoa
- Vitamin B12 ← shellfish, sardines, eggs

### 3. Biological Importance

Multiple methylation-dependent functions draw from the same donor-resource network rather than isolated nutrient streams. The same one-carbon resources support homocysteine remethylation, methyl-group transfer, membrane-related methylation context, and broader cellular maintenance. When donor availability falls, several PMs can lose capacity in parallel, making this KC a system-level bottleneck rather than a single-pathway issue.
